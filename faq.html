FAQ

Common questions.

Quick answers about self-hosting, configuration, and behavior.

Do I need to host this myself?

Yes. Flex Tracker is a self-hosted project. You run it on your own machine or a VPS so it can stay online.

Will it work in multiple servers at once?

Yes. One running bot instance can be in multiple Discord servers. Each server can configure its own alert channel, threshold, and polling.

What’s the best way to reduce spam?

Use one dedicated alert channel. Keep the roster tight and avoid polling too frequently.

Why do some commands reply privately?

Some slash commands respond ephemerally (private reply) so your server channels don’t become command logs.

Is this affiliated with Riot?

No. Flex Tracker is fan-made and not endorsed by Riot Games.
