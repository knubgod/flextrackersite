About

A practical bot for flex servers.

Built to answer one question quickly: “Are we stacked right now?”

What Flex Tracker does

Flex Tracker monitors a roster of League of Legends accounts and posts an alert when a threshold of roster players are detected queued together in Ranked Flex.

It’s designed to be low-noise: one clean alert at the right moment, plus optional status/help commands for server admins.

Why it’s self-hosted

This project stays self-hosted so you control uptime, configuration, and API usage. You can run it on a homelab for private servers, or on a VPS for consistent uptime.

The install guide includes Linux (pm2) and Windows options.

Project values

Predictable alerts, clean channels, responsible API behavior.

Clean output

Alerts are readable and posted in one place — not sprayed across the server.

Configurable

Adjust channel, threshold, and polling so it matches your community.

Self-owned uptime

You run it. Your bot stays online as long as your host stays online.
