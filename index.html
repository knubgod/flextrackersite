Ranked Flex

Stack alerts, match tracking, and clean server summaries. Know when your roster queues together — without the hassle.

Flex Tracker watches a roster of players and posts an alert when a stack is detected. Built for real servers: clear notifications, simple setup, and low noise. This is an open source project built on Discord (dot) js.
